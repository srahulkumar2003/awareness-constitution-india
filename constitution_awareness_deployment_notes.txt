PORT=8080
DB_URL=jdbc:mysql://${{MySQL.MYSQLHOST}}:${{MySQL.MYSQLPORT}}/${{MySQL.MYSQLDATABASE}}?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC
DB_USERNAME=${{MySQL.MYSQLUSER}}
DB_PASSWORD=${{MySQL.MYSQLPASSWORD}}
GEMINI_API_KEY=YOUR_NEW_GEMINI_API_KEY
GEMINI_MODEL=gemini-2.5-flash
CORS_ALLOWED_ORIGINS=https://awareness-constitution-india.vercel.app,http://localhost:5173

Important:
Each variable must be on a separate line.
Do not put all variables in one line.
Do not use DB_URL=mysql://...
Spring Boot needs DB_URL=jdbc:mysql://...

If private MySQL host has connection issues, use TCP proxy format:

DB_URL=jdbc:mysql://${{MySQL.RAILWAY_TCP_PROXY_DOMAIN}}:${{MySQL.RAILWAY_TCP_PROXY_PORT}}/${{MySQL.MYSQLDATABASE}}?useSSL=false&allowPublicKeyRetrieval=true&serverTimezone=UTC

Keep:

DB_USERNAME=${{MySQL.MYSQLUSER}}
DB_PASSWORD=${{MySQL.MYSQLPASSWORD}}
