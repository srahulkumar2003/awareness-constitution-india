# Resume Showcase Points

## Project Title
Awareness on Constitution of India | React.js, Spring Boot, MySQL, Chart.js, Framer Motion

## Resume Bullet Version
- Built a Constitution awareness web platform to educate users about Fundamental Rights, Duties, DPSP, Judiciary, constitutional values and citizen responsibilities.
- Designed an Interactive Constitution Book with open-book UI, chapter selection, Article references, Supreme Court case mapping, real-life examples and visual learning diagrams.
- Added gamification features including Citizen Score, badges, Constitution Champion levels and daily civic learning challenges to improve user engagement.
- Implemented a free, offline Constitutional Tutor using curated data instead of paid AI APIs, with explanations, Article references, case laws, follow-up questions and learning paths.
- Added multilingual learning support for English, Hindi, Telugu, Tamil and Kannada to improve accessibility for diverse users.
- Developed an analytics dashboard using Chart.js to visualize survey participation, rights awareness levels, state-wise awareness and age-group analysis.

## Interview Explanation
This project is a civic-tech education platform built to make the Indian Constitution easier to understand for students and citizens. The main feature is an Interactive Constitution Book where users can explore topics like Fundamental Rights, DPSP and Judiciary with Article references, case laws, real-life examples and diagrams. I avoided paid AI APIs and built a free offline tutor using curated constitutional data. The project also includes gamification, multilingual support, surveys and an analytics dashboard, making it practical for workshops, seminars and community outreach.

## Free Cost Build Strategy
- Frontend hosting: Netlify or Vercel free tier.
- Backend hosting: Render or Railway free tier, optional.
- Database: MySQL locally, Railway free tier, or static JSON for demo mode.
- AI alternative: rule-based offline tutor using curated data, no API key needed.
- Analytics: Chart.js with survey data from local JSON or backend endpoints.
